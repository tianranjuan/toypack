一个玩具webpac看，学习webpack时配套的演示项目。

支持基于webpack.config.js的配置。

实现了简单的loader、plugin、打包构建输出等功能。


运行方法：

在根目录下执行 `npm run toypack`，就可以在dist文件夹中查看构建后的输出了。

如果要修改代码直接在src文件夹下修改代码，然后执行 `npm run toypack`，查看构建后的结果变化。
